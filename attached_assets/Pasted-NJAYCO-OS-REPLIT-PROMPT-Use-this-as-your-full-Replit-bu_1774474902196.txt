NJAYCO OS REPLIT PROMPT

Use this as your full Replit build prompt:
Build a production-quality web application called NJAYCO OS for NJAYCO.com, the official digital headquarters of The Najee Jeremiah Company (NJAYCO). The application should feel like a person is turning on a Windows XP computer, but the entire operating system is actually a branded interactive website for a conglomerate. The site must be immersive, nostalgic, smooth, premium, and functional, not cheesy. It should combine retro desktop-computer aesthetics with modern web performance and responsive behavior.
Core Product Vision
NJAYCO OS is a Windows XP-inspired operating system simulation that serves as the public-facing website and portfolio hub for The Najee Jeremiah Company. Visitors should feel like they are entering the computer desktop of a multimedia, education, technology, logistics, and music conglomerate. Every company division should appear as an app, shortcut, browser page, folder, or document within the desktop operating system.
The website should function as:
* an immersive brand experience
* a desktop-style navigation system
* a holding company website
* a live product launcher
* a music storefront and media player
* a repository of division business plans
* an admin-controlled content system
The site should be highly visual and interactive, but still useful and understandable for real visitors, partners, fans, clients, investors, and customers.
